## 出力
### 文字列

### 数字
```
print(2)
→２
```

## ブール演算子
複数の条件があるときにそれが満たされているかを評価するために使う

### and
```
a and b
```
論理積‥aとbのどちらもTrueならTrue

### or
```
a or b
```
論理和‥aとbのどちらかがTrueならTrue

### and
```
not a
```
否定‥aがTrueならFalse,FalseならTrue
